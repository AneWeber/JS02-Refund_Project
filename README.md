# Refund Project
Refund is a project designed to list expenses for reimbursement requests.

![image](https://github.com/user-attachments/assets/22e67b90-91a5-4bd7-adf5-687734d165a8)

## Features

- Title

- Expense category

- Amount

- Able to delete

- Show summary with total amount and list of all expenses
